# 2026-05-26 — MVP: M0–M11

The first build arc: from empty repo to a working save manager.

## What landed
- **M0** Scaffold: Wails project, deps, domain model + SQLite schema.
- **M1** Launcher scanners: Steam, GOG, Epic, EA, Ubisoft, Xbox, Riot, Battle.net.
- **M2** Heuristic disk scan + pirate/repack detection.
- **M3** Metadata: SteamGridDB + Steam CDN + extracted exe icons.
- **M4** Ludusavi manifest matcher.
- **M5** Backup engine (versioned ZIP + sidecar manifest).
- **M6** Restore engine (pre-restore auto-backup).
- **M7** Game launcher.
- Dashboard UI (tiles + game drawer); build/run/smoke-test/fix.

## Hardening that immediately followed
- Path-scored matcher + manual override; reverse scan + canonical naming + grayscale "save-only" tiles.
- Duplicate-card and tokenizer fixes (Alan Wake II, DeathStranding).
- Pirate save scan (per-game + orphan); backup/restore reconcile.
- First push to GitHub `main`; auto-update + GitHub Actions release CI.
- Secret audit (SteamGridDB key out of source, CI-injected; manifest AWS-creds sanitized); sidebar shows the real version.

## State at end of phase
A working, self-updating Windows app that scans, matches, backs up, restores, and launches games, with
cover art and a dashboard. Releases flow through CI on tag push.
